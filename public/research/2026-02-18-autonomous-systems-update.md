## 2. Approval System: Configuration Lessons Learned

### The Challenge
- Initially attempted to use `approvals.exec` with `mode: "targets"` to auto-approve meta-agent updates from your Telegram account
- However, setting `enabled: false` should have disabled approvals entirely, but the system still prompted for approval
- The gateway appeared to ignore config changes or cached state persisted

### Solution
- Explicitly set `tools.exec.ask: "off"` to disable ask prompts
- This overrides any approval requirement and allows exec to run without interaction
- **Recommendation:** For autonomous systems, keep `tools.exec.ask: "off"` and rely on `approvals.exec` only for selective human-in-the-loop scenarios
